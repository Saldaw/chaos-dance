# Chaos Dance
## 📖 О проекте

**Chaos Dance** — это ритм-игра, где игрок управляет пони, попавшей в безумный мир Дискорда. Хаос заставил танцевать всех вокруг, и только главный герой может противостоять этому, следуя ритму и избегая опасностей. Проект вдохновлён вселенной *My Little Pony* и механиками классических аркадных ритм-игр. Ваша цель — продержаться как можно дольше, точно нажимая на нужные кнопки в такт музыке, и в конце концов найти выход из искажённого мира.

---

## 🎮 Управление

| Действие            | Клавиша        |
|---------------------|----------------|
| Движение            | `WASD`         |
| Выбор               | `Enter`        |
| Выход               | `Alt + f4`     |
